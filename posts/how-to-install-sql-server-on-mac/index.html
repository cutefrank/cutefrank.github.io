How to Install SQL Server on Mac

September 11, 2018

在 Mac 上安裝 SQL Server

必要條件

  • 安裝 Docker
  • 硬碟6G
  • Docker 記憶體2G(網路上建議4G以上)

下載和執行容器映像檔

下載映像檔

docker pull microsoft/mssql-server-linux:2017-latest

執行容器

docker run -e 'ACCEPT_EULA=Y' -e 'SA_PASSWORD=<YourStrong!Passw0rd>' \
-p 1433:1433 --name sql1 \
-d microsoft/mssql-server-linux:2017-latest

附註：EULA 最終用戶授權協定

列出所有容器

docker ps -a

變更 SA 密碼

docker exec -it sql1 /opt/mssql-tools/bin/sqlcmd \
-S localhost -U SA -P '<YourStrong!Passw0rd>' \
-Q 'ALTER LOGIN SA WITH PASSWORD="<YourNewStrong!Passw0rd>"'

連接到 SQL Server

打開 Bash

docker exec -it sql1 "bash"

使用 sqlcmd 進行本機連線

/opt/mssql-tools/bin/sqlcmd -S localhost -U SA -P '<YourNewStrong!Passw0rd>'

建立測試資料庫

CREATE DATABASE TestDB
SELECT Name from sys.Databases
GO

新增資料表

USE TestDB
CREATE TABLE Inventory (id INT, name NVARCHAR(50), quantity INT)
INSERT INTO Inventory VALUES (1, 'banana', 150); INSERT INTO Inventory VALUES (2, 'orange', 154);
GO

查詢資料

SELECT * FROM Inventory WHERE quantity > 152;
GO

離開 sqlcmd 工作階段

QUIT

離開 Bash

exit

容器外連線

連線

sqlcmd -S localhost,1433 -U SA -P '<YourNewStrong!Passw0rd>'

若尚未安裝 sqlcmd，請依照下面指令安裝 [ODBC 驅動程式 17 for SQL Server]和 mssql-tools，再執行連線命令

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew tap microsoft/mssql-release https://github.com/Microsoft/homebrew-mssql-release
brew update
brew install --no-sandbox msodbcsql17 mssql-tools
YES
如果不想打 YES 可改成 ACCEPT_EULA=y brew install --no-sandbox mssql-tools

若之後要刪除[ODBC 驅動程式 17 for SQL Server]，記得執行以下指令：

odbcinst -u -d -n "ODBC Driver 17 for SQL Server"

刪除容器

sudo docker stop sql1
sudo docker rm sql1
