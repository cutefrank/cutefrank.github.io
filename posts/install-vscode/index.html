How to install Visual Studio Code on Mac

September 10, 2018

在 Mac 上安裝 Visual Studio Code

  1. 先去官網下載： https://code.visualstudio.com/

  2. 接著打開 zip 檔，把 app 檔放進 Applications 資料夾內

  3. 打開 Visual Studio Code，輸入 Command + Shift + P，開啟 command palette，輸入 shell command

  4. 輸入 code . 測試是否安裝成功
