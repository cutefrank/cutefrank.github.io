How to Install SQL Server on Mac

在 Mac 上安裝 SQL Server 必要條件 安裝 Docker 硬碟6G Docker 記憶體2G(網路上建議4G以上) 下載和執行容器映像檔 下載映像檔 docker pull microsoft/mssql-server-linux:2017-latest 執行容器 docker run -e 'ACCEPT_EULA=Y' -e 'SA_PASSWORD=<YourStrong!Passw0rd>' \ -p 1433:1433 --name sql1 \ -d microsoft/mssql-server-linux:2017-latest 附註：EULA 最終用戶授權協定 列出所有容器 docker ps -a 變更 SA 密碼 docker exec -it sql1 /opt/mssql-tools/bin/sqlcmd \ -S localhost -U SA -P '<YourStrong!Passw0rd>' \ -Q 'ALTER LOGIN SA WITH PASSWORD="<YourNewStrong!Passw0rd>"' 連接到 SQL Server 打開 Bash docker exec -it sql1 "bash" 使用 sqlcmd 進行本機連線 /opt/mssql-tools/bin/sqlcmd -S localhost -U SA -P '<YourNewStrong!
