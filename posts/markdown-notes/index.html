Markdown Notes

September 7, 2018

1. 緣起

Markdown 是由 John Gruber 開發的一種輕量級標記式語言，初始版本於2004年3月19日發表。

2. 常用功能

2.1 標題(Heading)

This is an H1

# This is an H1

This is an H2

## This is an H2

This is an H3

### This is an H3

This is an H4

#### This is an H4
This is an H5
##### This is an H5
This is an H6
###### This is an H6

2.2 程式碼區塊

輸入四個空格或是一個Tab

區塊引言

>區塊引言

2.3 字體

2.3.1 粗體字(Bold)

粗體字 普通字

**粗體字**

2.3.2 斜體字(Italic)

斜體字 普通字

*斜體字*

2.3.3 設定文字顏色

綠色的文字

<font color="green">綠色的文字</font>

2.4 LaTex

進入 http://latex.codecogs.com/ 之後，輸入所需公式，產生的網誌放入括號中，範例如下：

Markdown 語法：

![](http://latex.codecogs.com/gif.latex?\\frac{1}{1+sin(x)})

HTML 語法：

<img src="https://latex.codecogs.com/png.latex?%24%24x%3D%5Cfrac%7B-b%5Cpm%5Csqrt%7Bb%5E2-4ac%7D%7D%7B2a%7D%24%24" style="width:auto;" />
